Die verpflichtenden Rechtsnormen (الأحكام التكليفية)

Der verpflichtende Rechtsnorm (الحكم التكليفي) ist die Anweisung des islamischen Gesetzgebers (الشرع), die sich auf die Handlungen der Menschen entweder in Form eines Gebots (الاقتضاء) oder einer Wahlmöglichkeit (التخيير) bezieht.

Das Wort "اقتضاء" bedeutet eine Aufforderung zum Handeln oder Unterlassen.

Die verpflichtenden Rechtsnormen werden in fünf Kategorien unterteilt:

1. Die Pflicht (الواجب):

Dies ist eine Handlung, die für den Verpflichteten (المكلف) bindend ist. Wer sie ausführt, wird belohnt, und wer sie unterlässt, wird bestraft.

2. Die Empfohlene Handlung (المندوب):

Dies ist eine Handlung, für die der Handelnde belohnt wird, aber wer sie unterlässt, wird nicht bestraft.

3. Das Verbotene (المحرم):

Dies ist eine Handlung, die der Verpflichtete unterlassen muss. Wer sie dennoch begeht, wird bestraft.

4. Die Verpönte Handlung (المكروه):

Dies ist eine Handlung, deren Unterlassung belohnt wird, aber deren Begehung nicht bestraft wird.

5. Das Erlaubte (المباح):

Dies ist eine Handlung, die weder Belohnung für das Unterlassen noch Bestrafung für das Begehen mit sich bringt.
